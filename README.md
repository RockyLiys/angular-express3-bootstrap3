快速搭建Web环境 Angularjs + Express3 + Bootstrap3
========================

[快速搭建Web环境 Angularjs + Express3 + Bootstrap3]

## Install

```{bash}
apt-get install git
apt-get install node.js
npm install bower
git clone https://github.com/RockyLiys/angular-express3-bootstrap3.git
source env.sh
```

## Running

```{bash}
sh run.sh
```
